# Changelog

## [Unreleased]

## [0.1.0] - 2026-06-13

### Added

- Local Python CLI with `scan`, `sanitize`, `scaffold`, `validate`, and `demo`.
- Deterministic content checks and masked audit reports.
- Human review gate before Skill generation.
- Skill scaffolder and validator.
- Synthetic examples and automated tests.
- Lint, type, build, CI, and repository security checks.
- English and Traditional Chinese maintainer documentation.
- Contextual review checklist.
- Proposal workflow Pilot and reusable proposal Skill.
- Fixed proposal output schema and structural test.

### Known limitations

- Pattern matching cannot identify every contextual disclosure risk.
- Human review quality cannot be verified automatically.
- Proposal evaluation currently checks structure rather than semantic quality.
- Core commands do not include cloud processing or telemetry.
