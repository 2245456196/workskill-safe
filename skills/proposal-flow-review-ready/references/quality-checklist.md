# Proposal Quality Checklist

- [ ] Decision-maker and expected decision are explicit
- [ ] Problems are separated from proposed features
- [ ] Stakeholder value and objections are differentiated
- [ ] MVP has measurable input and output
- [ ] Commercial assumptions are clearly labeled
- [ ] Claims have evidence status
- [ ] Missing data is visible
- [ ] Reviewer feedback changed structure where needed
- [ ] Slide outline uses one judgment per page
- [ ] No identifying context remains
