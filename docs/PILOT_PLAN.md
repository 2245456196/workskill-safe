# Private Pilot Plan

## Objective

Test whether a real work process can become a reusable Agent Skill without publishing confidential source material.

## Recommended pilots

1. sanitized meeting notes to action register
2. sanitized proposal process to review checklist
3. sanitized daily work log to reusable intake workflow

## Rules

- Keep source material in a private local workspace.
- Never commit raw excerpts.
- Use the scanner and sanitizer before scaffolding.
- Manually remove contextual identifiers.
- Test the generated Skill on a separate synthetic case.
- Publish only aggregate counts, synthetic reproductions, rule changes, Issues, and Releases.

## Public metrics

- pilot run count
- finding categories
- rejected case count
- false positives corrected
- false negatives corrected
- rules added
- Skills generated and validated
- external tester count
