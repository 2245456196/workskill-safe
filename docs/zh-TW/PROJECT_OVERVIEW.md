# WorkSkill Safe 專案總覽

## 一句話定位

WorkSkill Safe 是一套本機優先、隱私優先的開源 CLI，協助知識工作者將工作筆記中的可重複方法轉換成 Agent Skill 草稿，同時降低公司機密、個資與認證資訊被公開的風險。

## 這個專案解決什麼問題

工作方法常藏在會議紀錄、工作日誌、提案草稿與專案筆記裡。這些內容可能混有公司內部資訊、客戶名稱、個資、內部路徑、商業數字與認證資訊。

```text
私人筆記
  ↓
掃描敏感資訊
  ↓
去識別化與稽核報告
  ↓
人工複核
  ↓
生成並驗證 Agent Skill
```

## MVP 功能

| 指令 | 用途 |
|---|---|
| `scan` | 偵測 Email、電話、網址、路徑、認證格式與機密標記 |
| `sanitize` | 以類型化標記取代敏感內容，另產生遮罩稽核報告 |
| `scaffold` | 將已清理且人工確認的內容轉成 Skill 目錄骨架 |
| `validate` | 檢查 Skill 格式、Metadata 與剩餘隱私風險 |
| `demo` | 用完全虛構資料跑完整流程 |

## 使用邊界

工具只能降低明顯風險，不能保證完全匿名、符合所有法規或識別所有上下文機密。最容易忽略的是組合識別，產業、日期、地區、金額與特殊流程仍可能讓人猜出真實對象。
