# WorkSkill Safe 專案總覽

## 一句話定位

WorkSkill Safe 是一套本機優先、隱私優先的開源 CLI，協助知識工作者將工作筆記中的可重複方法轉換成 Agent Skill 草稿。

## 這個專案解決什麼問題

工作方法常藏在會議紀錄、工作日誌、提案草稿與專案筆記裡。WorkSkill Safe 提供一條可檢查、可複核的轉換流程：

```text
私人筆記
  ↓
內容掃描
  ↓
去識別化與稽核報告
  ↓
人工複核
  ↓
生成並驗證 Agent Skill
```

## MVP 功能

| 指令 | 用途 |
|---|---|
| `scan` | 掃描需要人工確認的內容 |
| `sanitize` | 以類型化標記取代內容，另產生遮罩稽核報告 |
| `scaffold` | 將已清理且人工確認的內容轉成 Skill 目錄骨架 |
| `validate` | 檢查 Skill 格式與剩餘風險 |
| `demo` | 用完全虛構資料跑完整流程 |

## 使用邊界

工具只能降低明顯風險，不能取代人工判斷。產業、日期、地區、金額與特殊流程的組合，仍可能讓外部人員推測原始情境。
