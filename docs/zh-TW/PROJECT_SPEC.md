# MVP 規格說明

## 使用者需求

身為知識工作者，我希望從私人工作筆記中抽取可重複的方法，但不把公司、客戶或個人資訊帶進公開的 Agent Skill。

## 功能驗收

- 掃描 Email、電話、網址、Windows／Unix 路徑、常見認證格式、機密標記與私人禁用詞。
- 預設不得覆寫原始檔。
- 產生類型化替換與遮罩稽核報告。
- 只有人工確認後才能生成 Skill。
- 驗證 YAML front matter、`name`、`description`、隱私章節與剩餘敏感格式。

## 技術限制

- Python 3.11+
- 核心功能離線執行
- 不依賴外部模型或服務
- 輸出可重現
- 支援繁體中文內容
- 無 Telemetry
